# AgroMonitor

AgroMonitor-Web es un prototipo académico de aplicación web pensado para optimizar la gestión agrícola. Permite a los usuarios registrar fincas, gestionar cultivos y recibir alertas y recomendaciones. Aunque el prototipo no implementa la integración real con sensores IoT ni Machine Learning, la estructura y la interfaz están diseñadas para futuras integraciones.

El prototipo está centrado en la experiencia de usuario y navegación, simulando cómo funcionaría la plataforma de manera real. Puedes probarlo en línea aquí: **https://agromonitor.azurewebsites.net/** 

# Tecnologías utilizadas:

ASP.NET Core MVC

Entity Framework Core

SQLite / SQL Server

HTML, CSS, JavaScript

C#
